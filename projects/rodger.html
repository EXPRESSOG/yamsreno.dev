Rodger


About


Rodger is a discord bot that I have been working on and off on for over a year now, the bot is meant to be a mee6 like bot but that is all text-based with no web control and with some other added features. The bot has moderation utilities, music, fun, information-based commands, and configurable greeting, prefix, farewell, logging, and the enabling/disabling of command categories. You can see Rodger in action on my dev server, invite to the server is on this Page.
